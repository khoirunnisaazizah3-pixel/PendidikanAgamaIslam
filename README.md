# 📖 Media Pembelajaran Interaktif — Pendidikan Agama Islam (PAI)

Kumpulan media pembelajaran berbasis web interaktif untuk mata pelajaran **Pendidikan Agama Islam (PAI)**, dirancang agar siswa dapat belajar secara mandiri, menarik, dan menyenangkan.

---

## 📁 Daftar File

| File | Topik | Jenjang |
|------|-------|---------|
| `puasa-interaktif.html` | Ibadah Puasa dalam Islam | SD / SMP |
| `qada-qadar-interaktif.html` | Iman kepada Qada dan Qadar | Kelas 6 SD |

---

## 🌟 Fitur Umum

Setiap media pembelajaran dilengkapi dengan:

- **Ayat Al-Qur'an** — tulisan Arab beserta terjemahan sebagai landasan materi
- **Navigasi Tab** — berpindah antar topik tanpa reload halaman
- **Materi Terstruktur** — penjelasan bertahap dari pengertian, jenis, hingga hikmah
- **Kuis Interaktif** — soal pilihan ganda dengan feedback langsung, skor, dan predikat hasil
- **Desain Responsif** — tampil optimal di smartphone, tablet, maupun komputer
- **Tanpa Instalasi** — cukup buka file `.html` di browser, langsung bisa digunakan

## 🚀 Cara Penggunaan

### Untuk Siswa
1. Unduh file `.html` yang diinginkan
2. Buka menggunakan browser (Chrome, Firefox, Edge, Safari)
3. Navigasi antar tab untuk membaca materi
4. Kerjakan kuis di tab terakhir untuk menguji pemahaman

### Untuk Guru
1. File dapat langsung dibagikan ke siswa melalui WhatsApp, Google Classroom, atau media lain
2. Dapat ditampilkan di proyektor/layar kelas langsung dari browser
3. Tidak memerlukan koneksi internet setelah file diunduh *(kecuali untuk memuat font Google)*
